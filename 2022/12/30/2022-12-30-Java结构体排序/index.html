Java结构体排序

2022/12/30 ACM ACM Java

java中的两种结构体排序方法

一、Comparable

强行对实现它的每个类的对象进行整体排序，实现此接口的对象列表（和数组）可以通过 Collections.sort 或 Arrays.sort 进行自动排序。

1.导入模块

import java.util.Arrays;

2.继承接口comparable+重写compareTo()方法

class node implements Comparable<node>{
    int x, y;
     @Override
    public int compareTo(node a) {
        if(this.x == a.x)
            return this.y - a.y; //按照 y 值从小到大排序
        else 
            return this.x - a.x;
    }
}

3.在Main函数中创建node数组并赋值

node[] arr = new node[n];
...

4.排序

Arrays.sort(arr, 0, n);  //排n个数，Arrays.sort(arr)则默认排全部
#例题：洛古P2240【深基12.例1】部分背包问题
题目描述

阿里巴巴走进了装满宝藏的藏宝洞。藏宝洞里面有 $N(N \le 100)$ 堆金币，第 $i$ 堆金币的总重量和总价值分别是 $m_i , v_i (1\le m_i,v_i \le 100)$。阿里巴巴有一个承重量为 $T(T \le 1000)$ 的背包，但并不一定有办法将全部的金币都装进去。他想装走尽可能多价值的金币。所有金币都可以随意分割，分割完的金币重量价值比（也就是单位价格）不变。请问阿里巴巴最多可以拿走多少价值的金币？

输入格式

第一行两个整数 $N,T$。

接下来 $N$ 行，每行两个整数 $m_i,v_i$。

输出格式

一个实数表示答案，输出两位小数

样例输入 #1 样例输出 #1
4 50
10 60
20 100
30 120
15 45		 240.00
代码样例：
import java.util.Arrays;
import java.util.Scanner;

class node implements Comparable<node>{
    int m;
    int v;
    double m_v;
    node(int m,int v){
        this.m=m;
        this.v=v;
        m_v=(double)v/(double)m;
    }
    @Override
    public int compareTo(node o) {
        node n=(node)o;
        if(m_v>n.m_v) {
            return -1;
        }else if(m_v<n.m_v){
            return 1;
        }else {
            return 0;
        }
    }
}

public class P2240 {
    
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner input = new Scanner(System.in);
        int n = input.nextInt();
        int t = input.nextInt();
        node[] arr =new node[n];
        for(int i=0;i<n;i++) {
            int a,b;
            a=input.nextInt();
            b=input.nextInt();
            arr[i] = new node(a,b);
        }
        Arrays.sort(arr, 0, n);
        int i=0;
        double value=0;
        while(t>0&&i<n) {
            if(t>arr[i].m) {
                value+=arr[i].v;
                t-=arr[i].m;
            }else {
                value+=arr[i].m_v*t;
                break;
            }
            i++;
        }
        System.out.printf("%.2f",value);
    }
}

二、Comparator

强行对某个对象 collection进行整体排序的比较函数，可以将 Comparator 传递给 Collections.sort 或 Arrays.sort 。

1.导入模块

import java.util.Arrays;
import java.util.Comparator;

2.创建结构体

class node{
    int x, y;
}

3.创建类mycmp继承接口Comparator

class mycmp implements Comparator<node>{
    public int compare(node o1, node o2) {
        if(o1.x == o2.x) 
            return o1.y-o2.y;    //x值相等时，按照y值从小到大排序
        else
            return o1.x-o2.x;   //按照x值从小到大排序
    }
}
#以上三步均在Main函数外

3.在Main函数中创建node数组并初始化

node[] arr = new node[n];
...
//类对象必须初始化，分配空间

4.排序

Arrays.sort(a, 0, n, new mycmp());
